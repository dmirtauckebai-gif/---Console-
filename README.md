Интернет-дүкен (Console)

🎯 Мақсаты

Бұл жоба интернет-дүкеннің негізгі жұмыс логикасын модельдейді.
Пайдаланушы тауарларды көреді, себетке қосады және сатып алу процесін орындайды.
Жоба арқылы бірнеше класспен жұмыс істеу, объектілерді басқару және қарапайым бизнес-логиканы құру дағдылары дамиды.

⸻

📌 Сұрақтарға жауап

1. Себет қалай ұйымдастырылған?

Себет (Cart) ішінде тауарлар тізім ретінде сақталады.
Әр тауар (Product) себетке қосылғанда, ол ArrayList<Product> ішіне жазылады.
Қажет болса, жалпы баға есептеледі.

⸻

2. Қандай кластар қолданылды?

Жобада 3 негізгі класс қолданылады:

* Product – тауар (атауы, бағасы, саны)
* Cart – себет (тауарларды сақтау, қосу, көрсету)
* User – пайдаланушы (аты, себеті)

Қосымша:

* Main – бағдарламаны іске қосу

⸻

3. Тауарлар қалай сақталады?

Тауарлар Product объектілері ретінде жасалады.
Олар:

* алдымен Main ішінде құрылады
* кейін User немесе Cart арқылы себетке жіберіледі
* ArrayList<Product> ішінде сақталады

⸻

📊 Қорытынды

Бұл жоба арқылы объектіге бағытталған бағдарламалау (OOP) принциптері қолданылды.
Негізгі идея – нақты өмірдегі интернет-дүкен моделін қарапайым консольдік жүйеде іске асыру.
Пайдаланушы тауар қосу, қарау және сатып алу әрекеттерін орындай алады.
